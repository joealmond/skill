## Format

Tasks should be formatted as:
```
- [ ] [P0] Critical task description
- [ ] [P1] High priority task
- [ ] [P2] Normal task (default)
- [x] Completed task
```

Priority levels:
- **P0**: Critical - blocking issues
- **P1**: High - important updates
- **P2**: Normal - routine maintenance
